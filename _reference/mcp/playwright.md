# Playwright MCP

**What it is:** Browser automation server for Claude Code.

**What it does:** Gives Claude a live browser — navigate, click, fill forms, take screenshots, inspect network requests, run JS. Used for scraping JS-rendered pages and UI testing.

**Config/data:** `.playwright-mcp/` at repo root (auto-managed, do not move)

**Key tools:** `browser_navigate`, `browser_click`, `browser_fill_form`, `browser_snapshot`, `browser_take_screenshot`, `browser_network_requests`
