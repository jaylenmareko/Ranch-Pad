# Workflows

Reusable step-by-step processes. Each file is a runbook — follow it top to bottom to complete the process.

---

## Index

| Workflow | Description |
|---|---|
| [Email Outreach](./email-outreach.md) | Run a cold email campaign end-to-end |
| [Web Scraping](./web-scraping.md) | Scrape contact or data lists from a target site |

---

## How to Add a Workflow

1. Create a new `.md` file in this folder (e.g. `roboflow-training.md`)
2. Use this structure:

```
# Workflow Name

## Purpose
What this process does and when to use it.

## Inputs
What you need before you start.

## Steps
1. Step one
2. Step two
...

## Outputs
What you end up with.

## Notes
Edge cases, gotchas, useful commands.
```

3. Add a row to the index table above.
